# 279.完全平方数
```
给你一个整数 n ，返回 和为 n 的完全平方数的最少数量 。
完全平方数 是一个整数，其值等于另一个整数的平方；换句话说，其值等于一个整数自乘的积。
例如，1、4、9 和 16 都是完全平方数，而 3 和 11 不是。
```
```java
class Solution {
    /**
     * 返回和为n的完全平方数的最少数量。
     *
     * @param n 目标整数
     * @return 和为n的完全平方数的最少数量
     */
    public int numSquares(int n) {
        // 创建一个长度为n+1的数组dp，用于存储从0到n每个数字可以由完全平方数相加得到的最少数量
        int[] dp = new int[n + 1];
        // 初始化一个较大的值作为dp数组的默认值，表示当前数字无法由完全平方数相加得到（或者需要很大的数量）
        // 这里选择n+1是因为如果某个数字无法由完全平方数相加得到，我们可以认为其需要的数量是n+1（一个比n大的数）
        // 这样在比较时，任何小于n+1的数都会被认为是更优的选择
        int max = n + 1;
        Arrays.fill(dp, max);
        // 初始化dp[0]为0，因为和为0不需要任何完全平方数
        dp[0] = 0;
        // 遍历所有可能的完全平方数（从1的平方到sqrt(n)的平方）
        for (int i = 1; i * i <= n; i++) {
            // 对于每个完全平方数i*i，更新所有可以由它加上前面的完全平方数得到的和j
            for (int j = i * i; j <= n; j++) {
                // 更新dp[j]为dp[j - i*i] + 1和dp[j]中的较小值
                // dp[j - i*i] + 1表示通过加上一个完全平方数i*i，从j-i*i得到j所需要的最少数量
                // 我们需要找到所有可能的i，使得dp[j]达到最小值
                dp[j] = Math.min(dp[j], dp[j - i * i] + 1);
            }
        }
        // 返回dp[n]，即和为n的完全平方数的最少数量
        // 如果dp[n]仍然是初始化的max值，说明n无法由完全平方数相加得到（但实际上在这个问题中不可能发生，因为总是可以选择n本身作为一个完全平方数）
        return dp[n];
    }
}
```
