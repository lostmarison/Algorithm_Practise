# 11.盛最多水的容器
```c++
class Solution {
public:
    int maxArea(vector<int>& height) {
        int left = 0, right = height.size() - 1;
        int result = 0;
        while (left < right) {
            int h = min(height[left], height[right]);
            // S=h*(right-left),(right-left)减小，h应增大
            result = max(result, h * (right - left));
            if (height[left] < height[right]) {
                ++left;
            } else {
                --right;
            }
        }
        return result;
    }
};`
